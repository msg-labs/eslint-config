# ESLint configuration

## Installation

```sh
npm install --save-dev @msg-labs/eslint-config
```

## Usage

if you already have an `.eslint` file, add the following line to your `.eslintrc` file:

```json
    "extends": "@msg-labs"
```

otherwise create a new one with this content

```json
{
    "extends": "@msg-labs"
}
```
